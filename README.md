## FINAL-EXAM-PROJECT

This project is a full-stack web application with user authentication using Node.js, Express, MongoDB for the backend, and a simple frontend with HTML, CSS, and JavaScript.

## Project Structure

final-exam-project/
│
├── backend/
│   ├── server.js           # Main backend server file
│
├── frontend/
│   ├── index.html          # Main HTML file
|   |── home.html   
|   |── about.html   
|   |── catalogue.html  
|   |── contact.html    
│   ├── styles.css          # CSS file for styling
│   └── script.js           # JavaScript file for frontend logic
|
├── package.json            # Backend dependencies and scripts
├── package-lock.json
|
└── README.md               # Project documentation
